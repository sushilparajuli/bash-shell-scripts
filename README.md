# Basic Operation

Displays all environment variables. If you want to get details of a specific variable, use `echo $VARIABLE_NAME`.

```bash
export
```

Example:

```bash
$ export
AWS_HOME=/Users/sushilparajuli/.aws
LANG=en_US.UTF-8
LC_CTYPE=en_US.UTF-8
LESS=-R

$ echo $AWS_HOME
/Users/sushilparajuli/.aws
```
